# Data structs
Data structs is a small module providing general data structures for everydays use.

## Included are (until now)

* Double linked lists
* FIFO Qeues
* LIFO Qeues (Stack)


## In work (planned)

* Heaps
* Binary-Trees
* Graphs (including DFS and BFS
## Installation
```bash
npm i @sebi2020/data_structs
```
