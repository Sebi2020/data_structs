# Data structs
Data structs is a small nodejs module providing general data structures for everyday use.

## Included are (until now)

* Double linked lists
* FIFO Queue
* Async Fifo
* LIFO Queue
* Array-Stack
* Heaps

## In work (planned)

* Binary-Trees
* Graphs (including DFS and BFS

## Installation
```bash
npm i @sebi2020/data_structs
```
